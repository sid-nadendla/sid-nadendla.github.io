Project Description and Expected Deliverables

The goal of this project is to learn how game theory techniques and tools can be designed/implemented to solve a real world problem. In this regard, two options are provided, and each student is expected to pick one of them.
  1. OPTION 1: This option is primarily designed for advanced graduate students and those who are interested in pursuing research in game theory. Interested students can pick a research problem inspired from any real-world application, and demonstrate how this real-world problem can be solved using game-theoretic techniques. Examples of such real-world problems can be found in the references below.
  2. OPTION 2: This option is primarily designed to improve implementation skills to implement bots based on games. Specifically, students are expected to implement Latent Tic-Tac-Toe (LTTT) environment in Python. LTTT is a modified version of the classical Tic-Tac-Toe, where moves are not revealed until after the opponent's next move, and lost if deemed invalid at the time of revelation. You are expected to model LTTT as an imperfect information game, and solve it using two algorithms, namely counterfactual regret-minimization (CFR) and Monte-Carlo CFR with outcome sampling (MCCFR-outcome). The performance of the two algorithms should be compared as shown in Figure 1 in [1]. Necessary reading material will be provided below in the references section.
Particularly, in Option 2, students may take inspiration from the implementation of classic Tic-Tac-Toe from Petting Zoo's library, which can be accessed using this link.
